# highload
Проектирую высоконагруженное приложение(TikTok)

## ДЗ1:  

Тип: платформа коротких видео с алгоритмической рекомендательной лентой

Ключевой функционал(MVP):

1) Персонализированная лента
2) Загрузка видео
3) Лайки
4) Комментарии
5) Подписки
